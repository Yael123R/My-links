# MyLinks - v1 (vibe coding)

## Input - Prompt utilizado
Crea una aplicación web similar a linktree que tenga botones 
para mis redes sociales y una sección para mi información de perfil.

## Output

### Stack generado
- HTML, CSS y JavaScript vanilla en un solo archivo index.html

### Estilos
- CSS puro sin librerías
- Layout: Flexbox centrado, no responsive
- Colores: sólidos, 3 colores (#f0f0f0, #333, #555)
- Animaciones: ninguna

### Recursos
- Iconos: ninguno
- Emojis: ninguno
- Imágenes: ninguna (placeholder con div gris)

### Lógica
- Vanilla JS (sin lógica, solo HTML estático)

## Tokens
- Input: 113 caracteres / 4 = ~28 tokens
- Output 1 (código): ~1800 caracteres / 4 = ~450 tokens
- Output 2 (respuesta chat): ~400 caracteres / 4 = ~100 tokens
- Total: ~578 tokens
